# Backend Ventas

## Descripción

Microservicio desarrollado con Spring Boot encargado de gestionar las operaciones relacionadas con las ventas del sistema mediante una API REST.

## Tecnologías

- Java
- Spring Boot
- Maven
- MySQL

## Instalación

```bash
git clone https://github.com/val4nti/Devops--back.git
cd Devops--back
mvn clean install
mvn spring-boot:run
```

## Funcionalidades

- Registrar ventas.
- Consultar ventas.
- Actualizar información.
- Eliminar registros.
